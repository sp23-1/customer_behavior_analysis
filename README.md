# customer_behavior_analysis
data analytics project showcasing customer behavior analysis using python, sql and  power BI

Customer Shopping Behavior Analysis

Tools: Python (Pandas, NumPy), SQL (MySQL), Power BI

Analyzed 3,900+ customer purchase records to understand revenue patterns and buying behavior.

Performed data cleaning, EDA, and feature engineering (age groups, loyalty segmentation).

Built SQL queries to examine discount effectiveness, shipping preferences, product ratings, and customer segments.

Developed a Power BI dashboard to visualize revenue trends, high-value customers, and subscription impact.

Delivered business insights and recommendations to improve customer retention, marketing targeting, and product strategy.
